# on-9game-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，不依赖特定后端服务或数据库，可直接在浏览器中打开浏览。

仓库内容主要为历史版本或特定用途的 HTML 文件集合，方便查阅、分享或离线使用。

## 目录说明

```
├── pages/          # 存放所有归档的 HTML 页面
│   ├── v1/         # 早期版本页面
│   ├── v2/         # 更新版本页面
│   └── ...
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选，用于快速导航）
└── README.md       # 本文件
```

- `pages/` 目录按版本或用途划分子文件夹，每个子文件夹内包含完整的 HTML 文件及其依赖。
- `assets/` 目录存放公共资源，避免重复引用。
- 根目录下的 `index.html` 仅用于提供页面索引，并非必须。

## 页面归档说明

- 所有页面均为纯前端 HTML，无服务器端依赖。
- 归档页面保留原始结构与功能，但可能因外部资源链接失效而无法完整显示。
- 部分页面可能包含已废弃的实验性功能，仅供参考。
- 不保证所有页面在所有浏览器或设备上完全兼容。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交问题或建议，但请注意不保证即时响应。
- 如需提交新的归档页面，请遵循现有目录结构，并在 Pull Request 中说明页面用途。
- 仓库不接受任何形式的商业推广或广告内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
